OpenAlgo Charts — examples

Build first: npm run build. Then open any example (a static file server helps with ES module imports).

1 · Panes & grid — HiDPI canvas layout, render loop, resize 2 · Static candles — DataLayer, scales, candle renderer, IST axes 3 · Interaction — pan, wheel-zoom, crosshair/magnet, history paging 4 · Live data — candle builder, live update, last-price line, volume pane 5 · Chart types — all standard styles via the chart-type registry 6 · Markers, events, EMA — buy/sell signals, E/D badges, indicator, price lines 7 · Transforms — Heikin Ashi, Renko, Range, Line Break, P&F, Kagi 8 · Trade layer (read-only) — order lines, position P&L, SL/TP bracket 9 · Chart trading — one-click, drag-to-modify, OCO, analyzer mode 10 · Depth ladder — 5→200 levels, virtualized, click-to-place 11 · Profiles & order flow — Volume Profile, TPO, Footprint

Every example is dependency-free and uses only the built dist/ bundles.
